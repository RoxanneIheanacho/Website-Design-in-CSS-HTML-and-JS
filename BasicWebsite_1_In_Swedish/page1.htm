  • A1 fråga: skriv allmänt vad ett protokoll är i datakommunikation.
  • A1 Svar: Protokoll gäller som regler som berättar hur en dator kommunicerar med en annan dator.
    Reglerna ger anvisning till hur och vart allt information från en ställe placeras till den nästa.
  • A2 fråga: I kommunikation på webben används bl.a. protokollen HTTP, TCP och IP.Beskriv vilken funktion dessa har.
  • A2 Svar

    HTTP: Hypertext Transfer Protocol gör det möjligt för att få webbsidor till webbläsare. En exempel skulle vara när man klickar i FirstClass för att öppna denna websida (page 3) och skicka den till internet explorer. Det läser HTML kod och gör det till en läsbar sida i IE genom HTTProtokol!
    TCP: TCP eller Transmission Control Protocol är en protokoll som jobbar med IP. TCP jobbar med att skicka meddelande inom en nätverk vid programmer som jobbar med hur dem utbyter data till varandra
    IP: Står för Internet Protocol som jobbar med regler om hur data paketer ska skickas och utbytes till dator.

    TCP / IP är en grupp som har kommunikations protokoll där det använder för att ansluta dator system i ett nätverk. Det är kallad efter två av protokollen i familjen: TCP, eller Transmission Control Protocol och Internet Protocol, eller IP. Hypertext Transfer Protocol som referreras som HTTP är också medlem av TCP och IP gruppen

    HTTP sker i en request från client ock server. När man anroppar en HTTP request då kan clienten och servern prata med varandra vid kod som anroppar.
